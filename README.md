# 🎬 Chatbot Netflix com IA

Um chatbot simples desenvolvido em Python utilizando LangChain e Groq AI para recomendar filmes e séries da Netflix de forma natural e interativa.

---

## 🚀 Tecnologias utilizadas

* Python
* LangChain
* Groq API
* Llama 3.3 70B
* Google Colab

---

## 📌 Funcionalidades

* Recomenda filmes e séries
* Sugere conteúdos parecidos
* Explica sinopses
* Responde perguntas sobre Netflix
* Chat interativo no terminal

---

## 🖥️ Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/SEUUSUARIO/NOME-DO-REPOSITORIO.git
```

---

### 2. Instale as dependências

```bash
pip install langchain langchain-groq langchain-community beautifulsoup4 lxml html5lib
```

---

### 3. Adicione sua chave da Groq

No código:

```python
os.environ["GROQ_API_KEY"] = "SUA_CHAVE"
```

---

### 4. Execute o projeto

```bash
python meu_chatbot.py
```
